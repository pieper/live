WebGPU LiveRenderer preview

A step-based volume ray-marcher written in TypeScript and running on your GPU via WebGPU — a browser port of the slicer-wgpu renderer, the same code that runs natively under Deno. This is the future rendering backbone for SlicerLive (replacing vtk.js); this preview shows a synthetic volume (a translucent soft-tissue shell with a dense inner blob).  ·  SlicerLive
